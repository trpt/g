# g
GnuPG wrapper in bash using zenity and rofi/dmenu

# Config
Inside script you can set up some things.

# Usage
`g action`

where `action` is:
`e` - encrypt message  
`d` - decrypt message  
`s` - sign message  
`se` - sign & encrypt message  
`ef` - encrypt file  
`df` - decrypt file  

Or simply run script to choose action

# Dependencies
bash  
zenity  
rofi or dmenu  

Tested in Arch Linux
